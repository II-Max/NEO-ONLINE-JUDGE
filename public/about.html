1. TÓM TẮT DỰ ÁN (ABSTRACT)

Mục tiêu: Xây dựng một nền tảng Web App giúp học sinh luyện tập lập trình đa ngôn ngữ (Python, C++) và nhận kết quả chấm điểm tự động ngay lập tức.

Điểm mới: Thay vì thuê máy chủ (VPS) tốn kém, dự án sử dụng kiến trúc Hybrid (Lai ghép): Dữ liệu lưu trên đám mây Google Firebase, còn việc xử lý code được thực hiện tại máy tính cá nhân thông qua Python Script.

Kết quả: Hệ thống hoạt động ổn định, chi phí vận hành bằng 0 đồng, hỗ trợ chấm song ngữ Python/C++, Chat thời gian thực và Bảng xếp hạng trực tuyến.

2. LÝ DO CHỌN ĐỀ TÀI

Trong chương trình giáo dục phổ thông mới, Tin học đã trở thành môn bắt buộc. Tuy nhiên:

  • Giáo viên tốn quá nhiều thời gian để chấm thủ công hàng trăm bài code của học sinh.
  • Học sinh thiếu môi trường thực hành có phản hồi ngay lập tức (biết đúng/sai ngay).
  • Các hệ thống chấm bài (Online Judge) hiện có thường phức tạp và yêu cầu chi phí hạ tầng cao.

=> Do đó, tôi phát triển hệ thống này nhằm giải quyết bài toán "Hiệu năng cao - Chi phí thấp".

3. CÔNG NGHỆ SỬ DỤNG

Frontend

HTML5, CSS3, JavaScript thuần. Giao diện Cyberpunk hiện đại, tối ưu trải nghiệm người dùng (UI/UX).

Backend Cloud

Google Firebase: Hosting (lưu web), Authentication (đăng nhập), Realtime Database (đồng bộ dữ liệu ms).

Core Judge

Đa Ngôn Ngữ: Hỗ trợ biên dịch cả Python 3 và C++ (G++). Sử dụng module `subprocess` để tạo Sandbox an toàn.

4. KIẾN TRÚC HỆ THỐNG (DATA FLOW)

Mô hình hoạt động theo cơ chế "Hộp thư thông minh":

Học Sinh
(Web Client) Firebase
(Hộp thư chung) Máy Chấm
(Python + C++)

Quy trình chi tiết:

  1. Học sinh chọn ngôn ngữ, nộp bài -> Web gửi code lên Firebase (trạng thái: pending).
  2. Máy chấm phát hiện tín hiệu -> Tải code về máy.
  3. Hệ thống tự động nhận diện ngôn ngữ để gọi trình biên dịch tương ứng (Python hoặc G++).
  4. Chấm điểm -> Đẩy kết quả ngược lên Firebase (trạng thái: completed).
  5. Web tự động cập nhật điểm số cho học sinh.

7. KẾT LUẬN & HƯỚNG PHÁT TRIỂN

Kết luận:

Dự án đã hoàn thiện trọn vẹn các tính năng đề ra. Hệ thống hoạt động ổn định với cả hai ngôn ngữ lập trình phổ biến nhất trong trường học là Python và C++.

Hướng phát triển:

  • Phát triển thêm tính năng "Phòng thi ảo" có đếm giờ.
  • Tích hợp AI (Gemini API) để gợi ý sửa lỗi code cho học sinh.
  • Mở rộng hỗ trợ thêm Java và Pascal.
